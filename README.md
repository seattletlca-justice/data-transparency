# Seattle Rental Housing Data Transparency

Public, replicable analysis of Seattle rental housing enforcement records.

## Purpose
This repository publishes:
- core KPI datasets used in the dashboard
- methodology and field logic used to calculate metrics
- dashboard/web pages that present findings to the public

## Principles
- Transparency: definitions and formulas are explicit
- Replicability: metrics can be reproduced from documented rules
- Accountability: public-facing claims map to concrete data fields

## Repository Structure
- `data/core/2026-03-30/` → canonical KPI tables
- `docs/methodology/` → dictionary, mapping rules, traceability docs
- `scripts/` → reproducible build scripts
- `site/` → website/dashboard files

## Data Scope
Current baseline includes records from 2003–2025 with KPI breakout focus for 2015–2025.

## Notes
Large raw source files are intentionally excluded from this public repo and retained separately.
